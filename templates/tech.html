What is Social Butterfly?

Social Butterfly allows you to anonymously chat with random strangers through Google Talk. If you use Gmail, you already have Google Talk.

Open Source

Social Butterfly is written in Python for Google’s App Engine cloud. You can view Social Butterfly’s clean, object-oriented source code.

Scalable

Social Butterfly uses techniques like memcache, sharding counters, and deferred task queues to scale to {{ stats.num_users }} users and counting.

Real-Time

Social Butterfly uses channels (similar to WebSockets) for real-time communication. The footer on every page is live yet scalable.
